# HyperCSI
- 這項專案是對3種物質的HyperCSI演算法來速度優化的版本，時間花費是原有演算法的60%~70%。主要是用三維空間的幾何特性來優化演算法的速度。

## 使用
- 主程式是**main_v2.m**；使用的data是**data10.mat**；HyperCSI演算法的function在**HyperCSI.m**內；使用上直接執行main_v2.m就會拿到三種指紋的圖片和三種物質的分布圖。
